如何使用Github（三）

这一节来学习如何使用Git Shell将线上的版本仓库下载到本地，并且建立版本分支。

将线上的仓库下载到本地(Fork)

我们为什么要把线上的仓库下载到本地呢？通常情况下是我们想要参与别人的项目。

Github上面有一个项目：https://github.com/octocat/Spoon-Knife

现在我们想要参与这个项目，需要进行以下步骤：

一、访问上面的链接，点击Fork按钮，

二、现在spoon-knife项目已经添加到你的线上版本仓库中了，我们要下载它。打开Git Shell，输入https://github.com/linjunyu/Spoon-Knife.git),这里的linjunyu是我注册的用户名，需要你改成你自己的

三、前两节我们把版本仓库建立在了D盘根目录，因此这里默认是在D盘目录的Shell，进入D盘，会发现多了一个spoon-knife目录，线上的源码已经下载下来了。

四、输入cd spoon-knife

五、输入git remote add upstream https://github.com/octocat/Spoon-Knife.git，这里我们使用的是原项目的地址，确保我们可以随时跟踪到原项目

六、输入git fetch upstream，获取原项目最新的与你本地版本库不同的修改，但是不修改你的本地文件

好了，这样我们就成功的将线上仓库下载到本地~

如何使用Git Shell建立版本分支(Branch)

如果你想要在原项目的基础上尝试一些新的思路或者开发新功能、修改等等，你可以建立原项目的一个分支，这样当进行修改时，原来的代码是不会受到影响的。

前几节建立了一个helloworld版本库，那么现在就来为它建立一个分支。

一、打开Git Shell，输入cd helloworld，进入本地的helloworld版本库

二、输入git branch test

三、输入git checkout test，将test分支设置为当前工作分支

四、输入git add ./，将helloworld目录的所有文件添加至上传队列

五、输入git commit -m 'a new test branch'，缓存提交

六、输入git remote add origin https://github.com/linjunyu/helloworld.git，配置上传目标仓库，这里的linjunyu是我的用户名，需要你改成你自己的，下面的linjunyu也要注意

七、输入git push -u origin test，将文件上传到test分支

然后访问：https://github.com/linjunyu/helloworld/branches，会发现多出了一个test分支！

好了，这样我们就成功的建立了版本分支~

这节教程就告一段落了，抽空更新第四节教程~

如何使用Github（一）

本系列教程转载于http://woiweb.net/github-gitshell-tutorial.html，然后经由我完善了一些内容。

Git是一个分布式的版本控制系统，最初由Linus Torvalds编写，用作Linux内核代码的管理。在推出后，Git在其它项目中也取得了很大成功，尤其是在Ruby社区中。

目前，包括 Rubinius和Merb在内的很多知名项目都使用了Git。Git同样可以被诸如Capistrano和Vlad the Deployer这样的部署工具所使用。

作为开源代码库以及版本控制系统，Github目前拥有140多万开发者用户。随着越来越多的应用程序转移到了云上，Github已经成为了管理软件开发以及发现已有代码的首选方法。

作为一个分布式的版本控制系统，在Git中并不存在主库这样的概念，每一份复制出的库都可以独立使用，任何两个库之间的不一致之处都可以进行合并。

GitHub可以托管各种git库，并提供一个web界面，但与其它像SourceForge或GoogleCode这样的服务不同，GitHub的独特卖点在于从另外一个项目进行分支的简易性。为一个项目贡献代码非常简单：首先点击项目站点的“fork”的按钮，然后将代码检出并将修改加入到刚才分出的代码库中，最后通过内建的“pull request”机制向项目负责人申请代码合并。

百度百科：http://baike.baidu.com/view/3366456.htm

使用Github来管理代码，看完这个系列的教程你会发现它的好处的。

开始动工

一、注册一个免费git帐号，可以无限建立访问权限为公共的版本仓库，注册地址：https://github.com/signup/free

二、依次填写，用户名、邮箱地址、密码，点击“creat an account”进行注册

三、点击右上方的”sign in”使用注册邮箱登录到Github

四、点击右上方的绿色按钮，+ New repository（新建一个版本仓库）

五、在“Repository name”一栏里填写版本仓库的名称，我们填写为“helloworld”，Description栏是描述，可填可不填

默认访问权限为公共，我们点击“Creating Repository”按钮，创建我们的第一个版本仓库。

接下来，我们需要在本地安装Github客户端。

六、以Windows为例，下载客户端：http://github-windows.s3.amazonaws.com/GitHubSetup.exe，安装完成后，会在桌面上出现两个图标，Git Shell和GitHub。

七、那两个图标分别是命令行工具和图形界面，我们先来学习使用Git Shell。

八、双击打开Git Shell,【win7以上版本的用户使用管理员权限打开】

九、输入git config --global user.name "linjunyu", 这里的linjunyu是我注册的用户名，需要你改成你自己的

十、输入git config --global user.email "346693350@qq.com", 这里的邮箱也是我的邮箱，需要你改成你自己的

十一、上面两步在客户端配置你的Git帐户，下面我们在本地建立自己的版本仓库，我把版本仓库建立在D盘。输入cd d:，切换到D盘目录。

十二、输入mkdir helloworld，建立一个helloworld目录，注意本地仓库名要和git中建立的仓库名一样

十三、输入cd helloworld，进入helloworld目录

十四、输入git init,初始化版本仓库

十五、输入touch README,建立一个README文件

十六、用文本编辑器打开README文件，随便输入一点文字，比如：this is a README file!

十七、输入git add README,将文件添加到上传队列

十八、输入git commit -m 'first commit',缓存提交，引号中的是对本次提交的描述，必须填写，不能为空 - -||

十九、输入git remote add origin https://github.com/linjunyu/helloworld.git,注意url格式，linjunyu是我的昵称，需要你改成你自己的，helloworld是刚才建立的版本仓库

二十、输入git push origin master，将README文件上传到版本库。这一步会一次让你填写你的git注册邮箱和密码，注意不要填错了

好了，到这一步，我们就完成了在线上和线下建立git版本库并上传文件的全部步骤。

如何安装Eclipse

如果你要使用Eclipse进行Java开发的话，要求你的电脑具有Java环境，所以应该先安装并配置好Java环境。

关于如何安装Java：http://linjunyu.com/2015/09/29/how-to-install-java/

安装Eclipse

一、首先进入Eclipse官方下载地址：http://www.eclipse.org/downloads/。

二、我们会发现下面Eclipse Package有很多个，这里我们选择Eclipse IDE for Java Developers。

三、我们根据自己电脑的系统位数，选择右边32bit/64bit，然后进入下载界面。

四、下载界面有些复杂，左下右有一栏Choose a mirror close to you，在这里我们可以选择最后一项China - Dalian Neusoft University of Information，从大连东软信息学院下载安装包。

五、之后进入了一个Donate界面，过几秒就会弹出Eclipse的下载框。

六、下载完成后，解压出来就是Eclipse的文件夹，双击文件夹中的eclipse.exe即可运行。

七、第一次打开应该需要设置workspace（你平时写的代码文件）的路径，选择默认的就可以。

然后就进入了Eclipse界面啦！

八、创建一个新项目：File->New->Java Project，Project name定为Helloworld吧！然后点右下角的“Finish”就可以。

并没有什么变化，把Welcome界面关掉，就看到了Eclipse的本体。

九、在Helloworld栏右击New->Class，创建一个类。

十、填写Name为Hello，然后点击“Finish”

终于，我们能愉快的敲代码了~

完结，撒花！

如何安装Java

如果你要使用Eclipse进行Java开发的话，需要你的电脑配置好Java环境，所以最开始应该先安装并配置好Java。

安装Java

Java JDK安装包，JDK是Java Development Kit的缩写，也就是开发工具包，里面内含平时用户用到的JRE，也就是Java Runtime Enviroment运行环境。所以装了JDK，JRE自然也被装上了。

一、JDK8官方下载网址是http://www.oracle.com/technetwork/java/javase/downloads/jdk8-downloads-2133151.html。

二、进入下载页面，下方有各个系统版本的JDK，根据自己的电脑系统选择合适的。32位系统安装32位JDK，64位系统64位JDK。（x86是32位，x64是64位）

三、下载下来之后，运行安装包，下一步下一步下一步…

四、安装完之后，还需要配置环境。

右击我的电脑，打开属性，左上角高级系统设置，打开“系统属性”，找到“高级”选项卡，找到右下角的“环境变量”。

点开“环境变量”

在环境变量中，要修改两个地方，一个是在系统变量中添加JAVA_HOME。选择“新建”，变量名填上JAVA_HOME，变量值填上JDK的安装目录 C:\Program Files\Java\jdk1.8.0_20（特别需要注意，jdk1.8.0_20这个版本是我电脑安装的JDK版本，你现在安装的JDK版本也许会比我的更加新，所以可以会是C:\Program Files\Java\jdk1.8.0_60或者更新，需要你到JDK的安装目录确定）

还要修改一个地方，就是Path，选择Path，然后点“编辑”，在最后面添加如下语句 ;%JAVA_HOME%\bin;

最后别忘了点“确定”，这样我们就安装并配置好了Java。

五、开始菜单搜索“cmd”，打开命令行，输入java -version这个命令，来检测我们的Java是否配置成功。

OK，系统找到了我们的Java版本，这样，我们的电脑就支持Java程序在电脑上运行，并且可以用Eclipse敲代码了。

关于如何安装Eclipse：http://linjunyu.com/2015/09/29/how-to-install-eclipse/
