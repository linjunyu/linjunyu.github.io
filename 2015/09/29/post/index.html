如何安装Eclipse

Eclipse需要你的电脑配置好Java环境，所以最开始应该先安装并配置好Java。

安装Java

Java JDK安装包，JDK是Java Development Kit的缩写，也就是开发工具包，里面内含平时用户用到的JRE，也就是Java Runtime Enviroment运行环境。所以装了JDK，JRE自然也被装上了。

JDK8官方下载网址是http://www.oracle.com/technetwork/java/javase/downloads/jdk8-downloads-2133151.html。

进入下载页面，下方有各个系统版本的JDK，根据自己的电脑系统选择合适的。32位系统安装32位JDK，64位系统64位JDK。（x86是32位，x64是64位）

下载下来之后，运行安装包，下一步下一步下一步…

安装完之后，还需要配置环境。

右击我的电脑，打开属性，左上角高级系统设置，打开“系统属性”，找到“高级”选项卡，找到右下角的“环境变量”。

点开“环境变量”

在环境变量中，要修改两个地方，一个是在系统变量中添加JAVA_HOME。选择“新建”，变量名填上JAVA_HOME，变量值填上C:\Program Files\Java\jdk1.8.0_20，因为在上面的安装过程中，我默认一直下一步安装，所以装在C盘，如果你在安装过程中改了，那可能是D盘或者E盘。同样变量值要做相应的更改。

还要修改一个地方，就是Path，选择Path，然后点“编辑”，在最后面添加如下语句;%JAVA_HOME%\bin;

最后别忘了点“确定”，这样我们就安装并配置好了Java。

开始菜单搜索“cmd”，打开命令行，输入java -version这个命令，来检测我们的Java是否配置成功。

OK，系统找到了我们的java版本，这样，我们的电脑就支持Java程序在电脑上运行，并且可以用Eclipse敲代码了。

安装Eclipse

首先进入Eclipse官方下载地址：http://www.eclipse.org/downloads/。

会发现下面Eclipse Package有很多个，这里我们选择Eclipse IDE for Java Developers。

根据自己电脑的系统位数，选择右边32bit/64bit，然后进入下载界面。

下载界面有些复杂，左下右有一栏Choose a mirror close to you，在这里我们可以选择最后一项China - Dalian Neusoft University of Information，从大连东软信息学院下载安装包。

然后就进入了一个Donate界面，过几秒就会弹出Eclipse的下载框。

下载完成后，解压出来就是Eclipse的文件夹，双击文件夹中的eclipse.exe即可运行。

第一次打开应该需要设置workspace（你平时写的代码文件）的路径，选择默认的就可以。

然后就进入了Eclipse界面啦！

创建一个新项目：File->New->Java Project，Project name定为Helloworld吧！然后点右下角的“Finish”就可以。

并没有什么变化，把Welcome界面关掉，就看到了Eclipse的本体。

在Helloworld栏右击New->Class，创建一个类。

填写Name为Hello，然后点击“Finish”

终于，我们能愉快的敲代码了~

完结，撒花！
