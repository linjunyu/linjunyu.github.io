Prim算法原理：
一个加权连通图，其中顶点集合为V，边集合为E；
(1)初始化：Vnew = {x}，其中x为集合V中的任一节点（起始点），Enew = {},为空；
(2)在集合E中选取权值最小的边，其中u为集合Vnew中的元素，而v不在Vnew集合当中，并且v∈V（如果存在有多条满足前述条件即具有相同权值的边，则可任意选取其中之一）；
(3)将v加入集合Vnew中，将边加入集合Enew中；
(4)重复(2)、(3)操作，直到Vnew = V，最终得到最小生成树。
